The Compendium

Our starting point is an Assay as defined in MIBBI (Minimal Information for Biological and Biomedical Investigations), which broadly corresponds to a MIACA file and to what is known as a Project in the MIDAS terminology. Here we use the term Compendium to refer to a set of data cubes arising from a common MIDAS file (and thus a common experiment(s)). Each Compendium is organized in a Matlab structure with fields,Name, Info, and data, e.g.: 

Compendium =
      Name:'MC-LGA-11111-CytoInh50'
      Info:'Data produced from Leonidas Alexopoulos in Feb 07 
            with Primary hepatocytes and HepG2s'
      data: [1x8 struct]
The Name usually contains the UID of the Compendium (see [1] and [2]). The Info field is a string with a brief documentation on the project.

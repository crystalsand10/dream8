Data import, storage, and handling in Matlab

In this section we will roughly describe how the data is handled in SBDataPipe. The user can interact either via Graphical User Interfaces (GUIs) or matlab scripting. To use the GUIs, no knowledge on Matlab programming is required, but if you want to write your own script, some knowledge will be convinient. Section 3 presents the GUIs, and Section 4 the matlab functions which can be called from the scripts.
